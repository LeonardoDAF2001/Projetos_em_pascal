Português:
Meu segundo projeto nessa linguagem. Nesse projeto foquei em desenvolver melhor minha lógica em estruturas de repetição, algoritmo de ordenação e criação de arquivo usando Pascal. 

A idéia:
Um programa simples que registra nome e quatro notas escolares de uma determinada quantidade de alunos. Com o programa o usuário poderá registrar, ordenar em ordem alfabética, 
exibir suas notas e média, e criar um arquivo .txt com todas essas informações.

Como funciona:
O usúario deverá digitar, já no início do programa, quantos alunos ele quer registrar. Em seguida é apresentado o menu principal com cinco opções, Registrar, Ordenar, Mostrar, Criar arquivo e Sair.
Na opção Registrar, o usuário deverá digitar o nome a ser registrado, em sequência as quartro notas desejadas.
Na opção Ordenar, o usuário ordena por ordem alfabética todos os alunos registrados.
Na opção Mostrar, o programa mostra todos os alunos e suas respectivas notas e médias (e se foi aprovado ou reprovado).
Na opção Criar Arquivo, o programa faz um arquivo .txt com as informações exatamente as mesmas da opção Mostrar.

Informações Adicionais:
- As notas e médias são números reais, ou seja, pode ser digitados números com casas decimais.
- As notas estão restringidas entre 1 a 10.
- O arquivo .txt se sobrepõe a cada vez que o usuário escolhe a opção quatro.







English:
My second project in this language. In this project, I focused on improving my understanding of looping structures, sorting algorithms, and file creation using Pascal.

The Idea:
A simple program that registers the names and four academic grades of a number of students. With this program, the user can register students, sort them alphabetically, display their grades and averages, 
and create a .txt file with all this information.

How it works:
At the beginning of the program, the user must enter how many students they want to register. Then, the main menu is displayed with five options: Register, Sort, Show, Make File, and Exit.
In the Register option, the user enters the student's name and then the four desired grades.
In the Sort option, the program sorts all registered students alphabetically.
In the Show option, the program displays all students along with their respective grades and averages (and whether they passed or failed).
In the Make File option, the program creates a .txt file containing exactly the same information shown in the Show option.

Additional Information:
- Grades and averages are real numbers, meaning decimal values can be entered.
- Grades are limited between 1 and 10.
- The .txt file is overwritten each time the user selects option four.
